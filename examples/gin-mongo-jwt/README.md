# gin-mongo-jwt

**Skill answers:** Gin · MongoDB · mongo-driver · JWT · RBAC=no · GraphQL=no · Docker=no

## What's here

Key files generated by the skill for this combination.

```
cmd/server/
  main.go            Entry point — loads env, connects DB, starts Gin
internal/handler/
  health.go          GET /health handler
go.mod               Module definition + dependencies
.env.example         Environment variables
```

## Run it

```bash
cp .env.example .env
# fill in MONGODB_URI and JWT_SECRET

go mod tidy
go run ./cmd/server/main.go
# → http://localhost:8080
# → http://localhost:8080/docs/index.html  (Swagger UI via swaggo)
```
