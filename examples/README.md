# Examples

These are representative samples of what the skill generates. Each example shows the key files for a specific framework + database + auth combination.

| Example | Stack |
|---|---|
| [`express-postgres-jwt/`](express-postgres-jwt/) | Express · PostgreSQL (Sequelize) · JWT · RBAC |
| [`fastapi-postgres-clerk/`](fastapi-postgres-clerk/) | FastAPI · PostgreSQL (SQLAlchemy) · Clerk |
| [`gin-mongo-jwt/`](gin-mongo-jwt/) | Gin · MongoDB (mongo-driver) · JWT |

These are not runnable as-is (they need a real database and `.env`), but they show exactly what the skill writes so you know what you're getting before you use it.
