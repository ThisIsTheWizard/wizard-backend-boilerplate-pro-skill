# express-postgres-jwt

**Skill answers:** Express · PostgreSQL · Sequelize · JWT · RBAC=yes · GraphQL=no · Docker=yes

## What's here

Key files generated by the skill for this combination. Not every file is shown — the full scaffold has ~40 files.

```
src/
  server.js          Entry point — starts HTTP server
  app.js             Express app factory — middleware + routes
  core/
    health.js        GET /health handler
    errorHandler.js  Global error handler + createError helper
.env.example         All environment variables with comments
package.json         Dependencies pinned to latest stable
```

## Run it

```bash
cp .env.example .env
# fill in DATABASE_URL and JWT_SECRET

npm install
npm run dev
# → http://localhost:3000
# → http://localhost:3000/docs  (Swagger UI)
```

curl -s -X POST http://localhost:3000/users/register \
